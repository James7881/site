Project Details

Using Jekyll and Bootstrap the interface for WATH required an easy handoff to a third party team with the intentions of integrating into Angular. Bootstrap and jQuery were requested for the convenience of additional developers to understand and manage the code base.

Process

This project was straight forward as all page designs were created in Sketch and used for reference. Scope requirements were clear and objectives discussed through a short Q&A session with Theron Studios.

Noteworthy

This project introduced a custom chart using Canvas as the foundation and was written about on our personal blog regarding the process and customization techniques titled “Creating A Custom Graph And Gradient With ChartJS.”
